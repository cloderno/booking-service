Проект

Hotel Booking Platform

Онлайн-платформа для владельцев отелей и клиентов.

Пользователи могут:

искать отели
смотреть номера
бронировать
оплачивать
оставлять отзывы

Владельцы:

добавляют свои отели
управляют номерами
следят за бронями

Администраторы:

модерируют контент
блокируют пользователей
просматривают статистику
